# TrackEase Prototype - Multi-Page Version

This is a separated multi-page version of the TrackEase prototype. Each section of the original single-page application has been split into its own page for better organization and navigation.

## Pages

- **index.html** - Dashboard with business setup and quick stats
- **inventory.html** - View and manage product inventory
- **add-product.html** - Add or edit products
- **sales.html** - Record sales and view recent transactions
- **reports.html** - Generate CSV reports and view sales summaries

## JavaScript Files

- **shared.js** - Common utilities, state management, and localStorage functions
- **dashboard.js** - Dashboard page logic
- **inventory.js** - Inventory page logic
- **add-product.js** - Product management logic
- **sales.js** - Sales recording logic
- **reports.js** - Reports and analytics logic

## How to Use

1. Open `index.html` in a web browser
2. Set up your business name and currency
3. Navigate between pages using the top navigation menu
4. All data is stored in your browser's localStorage

## Features

- ✅ Multi-page navigation
- ✅ Shared state across all pages
- ✅ Dashboard with quick stats
- ✅ Product inventory management
- ✅ Sales recording with recent sales view
- ✅ CSV export functionality
- ✅ Sales summary and analytics
- ✅ Mobile-first responsive design
