## ğŸ“¦ Urus Stok / Managing Stock

### Semak Stok / Check Stock:
1. Pergi ke tab **"Inventory"**
2. Lihat senarai semua produk dengan stok semasa / See all products with current stock
3. Produk dengan **stok rendah** akan ditanda / **Low stock** products are highlighted
4. **"Out of Stock"** untuk produk habis / **"Out of Stock"** for sold-out items

### Tambah Stok / Add Stock:
1. Tekan produk yang ingin dikemas kini / Tap product to update
2. Tekan "Adjust Stock"
3. Masukkan jumlah / Enter amount
4. Tekan "Add Stock"

### Kurangkan Stok / Remove Stock:
1. Tekan produk / Tap product
2. Tekan "Adjust Stock"
3. Masukkan jumlah / Enter amount
4. Tekan "Remove Stock" (untuk barang rosak/return / for damaged/returned items)

## ğŸ’³ Urus Hutang / Managing Credits

### Bila Pelanggan Berhutang / When Customer Buys on Credit:

1. Semasa checkout, pilih **"Pay Later"** / During checkout, select **"Pay Later"**
2. Jualan akan tersimpan sebagai "belum bayar" / Sale saved as "unpaid"
3. Untuk semak hutang / To check credits:
   - Tekan "Payments" dari Dashboard / Tap "Payments" from Dashboard
   - ATAU tekan banner oren di Reports / OR tap orange banner in Reports

### Bila Pelanggan Bayar / When Customer Pays:

1. Buka **"Outstanding Payments"**
2. Cari jualan yang berkenaan / Find the relevant sale
3. Tekan **"Mark as Paid"** / Tap **"Mark as Paid"**
4. Selesai! Rekod dikemas kini / Done! Record updated
