# TrackEase Assets

This directory contains the app icons and splash screen images.

## Required Assets

You'll need to create the following image files:

1. **icon.png** (1024x1024 px)
   - App icon shown on device home screen
   - Should be a simple, recognizable design
   - Suggested: A shopping cart or cash register icon with "TE" letters

2. **splash.png** (1284x2778 px for iOS, can scale down)
   - Shown when app is loading
   - Background color: #4CAF50 (green)
   - Suggested: TrackEase logo/name in white

3. **adaptive-icon.png** (1024x1024 px, Android)
   - For Android adaptive icons
   - Keep important content in center 66% of image
   - Background: #4CAF50

4. **favicon.png** (48x48 px, for web if needed)
   - Small icon for web browser tabs

## Temporary Solution

For now, you can use Expo's default icons or create simple colored squares:
- Use any image editing tool (Paint, Canva, Figma)
- Create a green (#4CAF50) background
- Add white "TE" text in the center
- Export at required dimensions

## Using Online Tools

Quick icon generators:
- https://www.canva.com/ (Free design tool)
- https://www.figma.com/ (Free design tool)
- https://appicon.co/ (Generate all sizes)

Once created, place the files in this directory.
